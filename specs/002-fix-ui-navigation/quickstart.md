# Quickstart: Fix UI and Navigation Issues

**Feature**: 002-fix-ui-navigation  
**Date**: 2026-04-09  
**Purpose**: Guide to running and testing the updated site.

## Prerequisites
- Node.js 18+
- npm or yarn

## Setup
1. Clone repository
2. `npm install`
3. `npm run dev` - starts Vite dev server
4. Open http://localhost:5173

## Testing
- Navigate to home: Check full viewport and fade-in animation
- Open navbar: Verify full screen, cross icon, navigation
- Visit pages: Ensure Guest List and Messages are functional

## Build
- `npm run build` - production build
- `npm run preview` - preview build
